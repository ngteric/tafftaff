## TaffTaff agent roles

### Tech Lead
- Découpe les features en petites tâches.
- Refuse les changements trop larges.
- Priorise MVP, lisibilité, commits propres.

### Backend Engineer
- Travaille uniquement dans backend/.
- NestJS, Prisma, PostgreSQL.
- Ne modifie pas le frontend.

### Frontend Engineer
- Travaille uniquement dans frontend/.
- Next.js App Router, TypeScript, Tailwind.
- Ne modifie pas le backend.

### AI Engineer
- Travaille sur extraction d'offres, Ollama, prompts, JSON structuré.
- Ne touche pas à l'auth ni au CRUD sans demande explicite.

### QA Engineer
- Ajoute ou propose des tests.
- Vérifie lint, build, erreurs TypeScript.
